Multiplicación y división

Multiplicación y división

Para multiplicar o dividir números enteros debemos calcular 2 resultados:

  1.  El número resultado de la multiplicación y división. Para ello multiplicamos o dividimos normalmente como haríamos si los dos números fuesen positivos.
  2.  El signo del número resultante. El signo resultante lo podemos calcular a partir de la regla de los signos:

     

    + : + = +
    + : - = -
    -  : + = -
    - : - = +

Ejemplos:

 

(+4) · (+5) = (+20)
(+40)  : (-8) = (-5)
(-6)  · (+3) = (-18)
(-6)  : (-3) = (+2)
